# Jewar Airport Real Estate Client Analysis

## Project Overview
This project focuses on analyzing client interest in real estate properties near the upcoming Jewar Airport region. The objective was to understand buyer behavior, identify high-interest client segments, and create a structured dashboard for better decision-making.

The analysis was performed using Excel dashboards, client segmentation techniques, AI-assisted insights, and HTML-based reporting.

---

## Objectives
- Analyze how many clients are interested in buying property near Jewar Airport
- Identify demand patterns based on occupation
- Understand location-wise buyer distribution
- Segment clients according to preferences and investment interest
- Create visual dashboards for easier interpretation

---

## Tools & Technologies Used
- Microsoft Excel
- Pivot Tables & Charts
- Data Cleaning Techniques
- AI-assisted Analysis
- HTML Reporting
- Dashboard Design

## Dataset Information
The dataset includes:
- Client Occupation
- Preferred Location
- Property Interest Area
- Budget Range

## Key Analysis Performed

### 1. Occupation-Based Analysis
Analyzed which professions showed higher property buying interest near Jewar Airport.

### 2. Location-Wise Client Segmentation
Studied which cities and regions generated maximum leads and inquiries.

### 3. Buyer Interest Tracking
Compared client demand across different property categories.

### 4. Dashboard Visualization
Created an Excel dashboard to present insights in a simple and interactive format.
